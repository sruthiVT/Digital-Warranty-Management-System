 Digital Warranty Management System

 Overview

The Digital Warranty Management System is a web-based application that helps customers register products, manage warranty details, and submit warranty claims online.

 Features

* Product Registration
* Warranty Tracking
* Warranty Claim Management
* User Dashboard
* Product Listing

 Technologies Used

* HTML
* CSS
* JavaScript
* Salesforce (Apex, Objects, Flows)

Project Structure

* Frontend developed using HTML, CSS, and JavaScript
* Backend developed using Salesforce

 How to Run

1. Download the project.
2. Open `index.html` in a web browser.
3. Navigate through the pages using the menu.
